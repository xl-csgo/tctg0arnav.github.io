Code Highlighting#

Hugo will highlight fenced code blocks automatically:

package main

import "fmt"

func main() {
		fmt.Println("Hello, Hugo!")
}

Shortcode-based highlighting also works:

export const greet = (name = "world") => {
	console.log(`Hello, ${name}!`);
};
